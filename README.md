# pyDoS2

PyDoS2 is a DoS script, this scripts is a remastered version of pyDoS

## Installation

To install the script, follow the instructions bellow

```bash
sudo git clone https://github.com/AzgarDev/pyDoS2.git
cd pyDoS2
sudo chmod +x install.sh
./install.sh
```

## Usage

```python
python DoS.py
```
